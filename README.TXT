
Just some examples of personal lisp and python code done at home.

- Rick

├── lisp: contains some lisp modules I wrote that are imported by
          projects or used interactively.
					
└── python
    |       
    |   houseSpeechController.py: say time and weather from cron
    |
    |   cam-sort.py: archives images taken by webcam every few minutes
		                 tries to sort dark images into separate directory.
    |
    |   julia_fullscreen.py: draw julia set w/ Pygame

    ├── MonsterBattle: a game written with Python/PyGame (2006-2008)
		                   with a couple of friends. (older style Python)
											 
    └── neurota: more current personal project that implements a
                 micro-database with Python data structure for
		             personal notes.

                 bin/neurota - top level command
